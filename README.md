# nlp-emotion-detector
Streamlit web app for detecting emotions from text using a trained ML model.
# NLP Emotion Detector

A machine learning-based web app that detects emotions from text using Natural Language Processing (NLP).
Built with Python, Scikit-learn, and Streamlit.

# Live Demo
https://your-app-link.com

# Project Overview

This project analyzes text input and predicts the emotion behind it such as:

😊 Happy
😢 Sad
😡 Angry
😨 Fear
😐 Neutral

The model is trained on labeled text data and uses NLP techniques to understand patterns in language.

# Technologies Used
Python 
Pandas & NumPy
Scikit-learn
Natural Language Processing (NLP)
Streamlit (for UI)
Pickle (for model saving)
# Project Workflow
Data Collection 
Text Preprocessing (cleaning, tokenization) 
Feature Extraction (TF-IDF / Bag of Words)
Model Training (Logistic Regression / ML model)
Model Evaluation 
Web App using Streamlit 
# How to Run This Project
1️ Clone the repository
git clone https://github.com/FarahHanif723/nlp-emotion-detector.git
2️ Move into project folder
cd nlp-emotion-detector
3️ Install dependencies
pip install -r requirements.txt
4️ Run Streamlit app
streamlit run app.py
